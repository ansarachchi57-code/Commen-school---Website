System Notice (අඩුපාඩු/Shortcomings & Limitations)

  • 1. SMS / WhatsApp Messages: requires a real backend service (like Twilio or API) to actually send messages. Buttons currently show a mockup alert.
  • 2. PDF Generation: Currently mocked. In production, a library like `jsPDF` or a backend printing service is needed to generate formatted certificates/reports.
  • 3. Data Persistence (දත්ත ගබඩා කිරීම): This is a UI mockup. If you refresh, data typed in inputs will reset. A database backend (SQL/NoSQL) is needed.
  • 4. File Uploads: Selecting files works locally but they are not saved to a server.
